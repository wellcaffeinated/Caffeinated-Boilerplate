# Caffeinated Boilerplate

Mainly for personal use... but feel free to take.

Boilerplate using:

* RequireJS 2.0
* jQuery 1.7.2
* Stapes.js

Has a build configuration (that you should use).

```bash
  $ cd library/js
  $ r.js -o require-profile.js
```
